目标：通过自然搜索 + 转化优化，实现 **每月 300 个合格 RFQ 询盘**。

每月根据 Google Search Console（查询、页面、CTR）和 GA4（`generate_lead`、`form_start`）更新本文档。

## 月度工作流

1. **GSC** — 导出 Top 50 查询；分配到最近 URL；记录 CTR 与排名位置。
2. **内容缺口** — 有展示但无匹配页面的查询 → 新博客或扩展 Hub FAQ。
3. **内链** — 每篇新博客：2 条 Hub 内链 + 1 条 `/contact/`。
4. **索引** — T1 页面实质性更新后，请求重新索引。
5. **转化** — 追踪 `form_start` → `generate_lead` 漏斗；优化有流量但 RFQ 率低的页面。

- [ ] 每页 1 个主词 + 1–2 个次词，写在 Title/H1，不堆砌（1-15 关键词密度）
- [ ] T1 词与上文「主打关键词地图」一致；KE 只用于**筛词**，不用于算总流量
- [ ] 博客 tag / About 等页不当作商业守城词

**Meta / URL / Heading（1-9～1-13）**

- [ ] 全站 Title 含主词 + 品牌；Description 含 RFQ/24h 等转化话术
- [ ] URL 短、小写、连字符；面包屑与 Hub 内链一致（1-11、1-12）
- [ ] 每页 H1 唯一；H2/H3 覆盖次主题，不跳级

**内链与媒体（1-13～1-15）**

- [ ] 每篇新博客：2 条 Hub/T1 内链 + 1 条 `/contact/`（与「月度工作流」一致）
- [ ] 对比页 / 案例页 CTA 链到 `/contact/?source=…`
- [ ] 工厂 tour 视频在 About / 首页可播放（1-14 嵌入视频）

**质量与算法（1-20～1-26、1-22）**

- [ ] 图片有 alt、WebP、合理尺寸（1-23 图片 SEO）
- [ ] 无大段模板重复；薄页 noindex 或合并（2-6 重复度）
- [ ] 遵守 Helpful Content：有实测数字、有工程师视角，非纯 AI 堆砌

### 第 2 章 — 实操与转化（课程 2-x）

**搜索意图与用户（2-1～2-3）**

- [ ] 能力/产品/材料页按**采购意图**写（报价、公差、MOQ），非百科
- [ ] `/contact/` 当作最重要落地页优化（2-4）

**Checklist 与工具（2-5、2-9～2-17）**

- [ ] 跑通一遍站内 SEO Checklist（Meta、canonical、hreflang、schema）
- [ ] GA4 自然搜索 + 着陆页报告每周看一次

- [ ] 站点地图已提交；新 T1 改动后「请求编入索引」（2-14）
- [ ] （可选）SEMrush 看竞品 Top 词与外链差距（2-17、3-7）

**关键词系统布局（2-7）**

- [ ] Hub 覆盖类目词；详情页覆盖具体工艺/材料/零件词
- [ ] 不建与能力重复的薄 city 页；用 blog 区域指南代替

**转化（2-10、2-11）**

- [ ] 追踪漏斗：`comparison/selection` → `rfq_form_view` → `rfq_form_submit`

### 第 6 章 — 没询盘诊断（课程 1-21、6-6、94）

每月对照：

| 现象 | 优先动作（课程逻辑） |
|------|----------------------|
| GSC 几乎无展示 | P0：sitemap、索引、T1 Title |
| 展示涨、点击不涨 | 改 Title/Meta 提 CTR（1-9） |
| 点击涨、RFQ 不涨 | 信任（ISO/案例/视频）+ 联系页 FAQ（2-4、2-10） |
| 只有 blog/glossary 流量 | 加强内链到 T1 + compare（3-13） |
| KE 有量但 GSC 无词 | 正常：先等收录 3–6 月，别换词太勤 |

**流量预期（KE + 收录模型，非承诺）**

| 阶段 | 自然会话/月（粗估） | RFQ @1% CVR |
|------|-------------------:|------------:|
| 0–3 月 | 50–300 | 1–3 |
| 6 月 | 400–1,200 | 4–12 |
| 12 月 | 1,500–3,500 | 15–35 |
| 100 RFQ/月目标 | 需 ~8,000–10,000 会话 | 约 9–18 月 |
